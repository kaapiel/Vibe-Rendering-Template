Documentazione non trovata

La documentazione di riferimento online (Aiuto) non è stata trovata. Ciò può essere dovuto ad uno dei seguenti motivi:

L'Aiuto in linea non è stato installato

Il Sistema di Aiuto può essere installato in uno dei seguenti modi:

Update Online

Dapprima assicuratevi che la lingua nella quale il Sistema di Aiuto deve essere installato sia disponibile.


  1. Richiamate dal menu principale di CINEMA 4D Aiuto | Verifica Aggiornamenti

    • Nella lista che appare, guardate se il file Aiuto di cui avete bisogno è disponibile. Se non lo è, spostatevi nella tab "Optional” per vedere se l’opzione è disponibile.
    • Se il file è disponibile, attivate/controllate il file e continuate con l’installazione.
    • Se l’opzione non è disponibile, il Sistema di Aiuto non è disponibile nella lingua desiderata. Continuate a cercare per effettuare degli aggiornamenti in futuro.

  2. Aggiornamento Manuale
    Per effettuare manualmente l’aggiornamento del Sistema di Aiuto è necessario avere il file c4dupdate che corrisponde alla versione di CINEMA 4D in possesso salvato localmente o su un mezzo che può essere accessibile offline (installazione DVD, file di download).

    • Avviate CINEMA 4D e aprite Aiuto | Installazione Manuale dal menu principale.
    • Nella finestra di selezione che appare, selezionate il file c4dupdate e cliccate su "Apri".
    • Tutti i passi successivi sono come quelli delle opzioni della guida online.

Se l'aiuto in linea non è disponibile nella vostra lingua, sarà aperta automaticamente la versione inglese.

E' stato selezionato il percorso sbagliato

Il percorso del file d'aiuto può essere definito nel campo Percorso di Aiuto Locale, localizzato nel menu preferenze di CINEMA 4D (menu principale: Modifica / Preferenze - / File / Aiuto Percorso Locale).

Se il file d'aiuto non è localizzato nel percorso di default e nessun percorso è stato definito nel menu preferenze, non sarà aperto nessun file d'aiuto. Assicuratevi che il campo Percorso di Aiuto Locale sia vuoto o contenga il percorso appropriato.
