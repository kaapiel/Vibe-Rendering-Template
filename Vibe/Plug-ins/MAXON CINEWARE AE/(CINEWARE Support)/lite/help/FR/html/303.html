Function available in CINEMA 4D Prime, Visualize, Broadcast, Studio & BodyPaint 3D

La documentation est introuvable

La documentation de référence en ligne (l'Aide) est introuvable. Les deux principales raisons possibles sont les suivantes :

L'Aide n'est pas installée

Le système d’aide peut être installé de deux façons :

Mise à jour en ligne

Premièrement, assurez-vous que la langue dans laquelle l’aide doit être installée est disponible.


  1. Sélectionnez Aide|Vérifier les mises à jour dans le menu principal de CINEMA 4D
  2. Regardez dans la liste qui apparaît si l’option correspondante est disponible. Sinon, cliquez sur le menu "optionnel” pour voir si le fichier est disponible.
  3. Si le fichier est disponible, cochez-la et continuez l’installation.
  4. Si l’option n’est pas disponible, le système d’aide n’est pas encore disponible dans le langage désiré. Vérifiez ultérieurement si c’est le cas.

Mise à jour manuelle

Pour mettre à jour le système d’aide manuellement vous devez avoir le fichier correspondant à votre version de CINEMA 4D enregistré en local (DVD d’installation, fichier téléchargé).


  1. Démarrez CINEMA 4D et sélectionnez Aide|Installation manuelle dans le menu principal.
  2. Dans la fenêtre de sélection qui apparaît, sélectionnez le fichier c4dupdate et cliquez sur Ouvrir.
  3. Les étapes suivantes sont identiques à la mise à jour en ligne.

Si le système d'Aide n'est pas disponible dans votre langue, l'application active automatiquement sa version anglaise. Vérifiez que la version anglaise du système d'Aide est bien installée!

Le chemin d'accès à l'Aide est erroné

Le chemin d'accès aux fichiers d'Aide peut être défini dans le champ Chemin d'accès à l'Aide locale, dans la section Aide du panneau Préférences générales de CINEMA 4D (menu principal : Edition / Préférences/ Chemin d’accès à l’aide locale).

Normalement, aucun chemin d'accès ne doit être entré dans ce champ. Toutefois, une fois qu'un chemin d'accès est entré, CINEMA 4D va l'utiliser pour accéder aux fichiers d'Aide.

Si les fichiers d'Aide ne se trouvent pas à leur emplacement par défaut et qu'aucun chemin d'accès n'a été défini dans les préférences, aucun fichier d'Aide ne sera ouvert. Vérifiez que le champ Chemin d'accès à l'Aide locale est vide ou contient le chemin d'accès approprié.
